Основные атрибуты input

  • type - задает тип инпута, влияет на его отображение
  • name - задает ключ, нужен для бекенда, на отображение не влияет, нужен для корректного взаимодействия с radio
  • value - данные, введенные по умолчанию
  • placeholder - подсказка, пример того, что пользователь должен ввести
  • requred - означает, что поле обязательно заполнить, используется для валидации
  • pattern - содержит регулярное выражение для валидации
  • disabled - блокирует ввод
  • autocomplete - определяет, может ли браузер автоматически заполнить это поле на основании ранее введенных данных

Типы input

Автозаполнение

В большинстве браузеров автозаполнение для некоторых типов инпутов будет предлагаться автоматически, на основании значения атрибутов type и name

Стандартное автозаполнение можно запретить или разрешить для всей формы с помощью атрибута autocomplete со значениями on и off

Чтобы указать конкретный тип автозаполнения для инпута, используется атрибут autocomplete

Label

label - заголовок для элемента формы. Он должен всегда присутствовать в разметке, не должен быть скрыт с помощью свойства display.

Использование атрибута placeholder не отменяет необходимость использования label
